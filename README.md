# DataValidationApp
Data Validation App


Save the app.py in /users/<yourname>/DataValidationApp/app.py
Save the Install_DataValidationApp.command in the desktop
Double click the Install_DataValidationApp.command to install it 
From then on you can just doubleclick the Install_DataValidationApp.command to run the App

When the App opens - select an excel file to validate and click the validate button. 
The validation will happen and the results will be saved in the same file in a different tab
